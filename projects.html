Feb. 2020 - May 2020

K-Medians in a Directed Tree

Prof. Smruti Ranjan Sarangi, IIT Delhi
(Algorithms)

The project aims to find the optimal placement of cache proxies in a computer network maintaining minimum overall cost. We moduled a C++ program that computes k-medians in a directed tree (edges directed from child to parent) in O(Pk2).
It is implemented in 2 phases and each phase uses a space of O(nk) complexity that is better than already established O(n2k) space complexity. This method employs dynamic programming for efficiency whereas best known algorithm for undirected tree is NP-hard.

  • 'k' is the number of resources to be placed, 'P' is path length of tree and 'n' is the number of vertices in the tree

Feb. 2020 - May 2020

MIPS Architecture Processor

Prof. Preeti Ranjan Panda, IIT Delhi
(Computer Architecture)

Project based on the VHDL implementation of a module consists of a multi-cycle MIPS architecture processor on FPGA and piplelined simulation fo the processor considering variable delays and single-layer Cache memory. BlockRAM Memory Generation is used to store the data as well as machine code(initialized using coe file). Demonstrated on FPGA and executes a large subset of MIPS instructions. A state machine is used to implement the control of the processor.
To analyze the effiency of the module, a 5-stage pipelined MIPS C++ Simulation is implemented that removes data hazards by avoiding stalls via forwarding and bypassing. It analyzes the efficiency by varying HIT and MISS probabilities and variable delays for each memory access.
